Desafio
Realize uma análise completa usando Python e a biblioteca Pandas numa base de dados de preços de aluguéis em São Paulo.
Utilize a regressão linear para modelar uma relação entre variáveis do conjunto de dados.
Os estudantes aprenderão a importar dados, limpar, explorar, visualizar e tirar conclusões a partir dos dados,
além de ajustar um modelo de regressão linear e avaliar seu desempenho.
